üìä How Amplitude Processes This

When you create a chart grouped by "app users" (a user property), Amplitude looks for the user_properties object in the raw event data. If this object doesn't exist or doesn't contain the "app users" key, Amplitude cannot find a value and displays (none) in your chart.
